# Update Tweet

plo-game (BabyPLO) で main に取り込まれたアップデートを **X 用の下書き** にするスキル。コミットメッセージから1行ずつのユーザー目線サマリを起こし、そのまま貼れる箇条書き形式で出す。

## 入力

- 引数なし → 今日の日付で main 取り込み分を対象
- `M/D` または `YYYY-MM-DD` → その日の main 取り込み分
- `since=<ref>` (例: `since=2026-05-05`、`since=6dbd065`) → 指定 ref から `origin/main` HEAD までの取り込み分

## データ取得

`git` のみ使用。DB やネットワーク呼び出しは不要。

```bash
# 最新化
git fetch origin --quiet

# 日付指定 (例: 2026-05-07)
git log origin/main --no-merges \
  --since="2026-05-07 00:00" --until="2026-05-07 23:59" \
  --pretty=format:'%h %ci %s'

# since 指定
git log <ref>..origin/main --no-merges --pretty=format:'%h %ci %s'
```

`--no-merges` で `Merge pull request ...` のマージコミットを除外する。本文も見たい場合は `git show --stat <hash>` で個別に取る。

## Workflow

### Step 1: 対象コミットの収集

上記コマンドでハッシュ・日時・サブジェクトを列挙。リスト化する前に、**ユーザー(プレイヤー)に見える変更だけ**を残す:

- 残す: 機能追加、UI 変更、ゲーム挙動の修正、バランス調整
- 落とす: 内部リファクタ、テスト追加、ドキュメント、依存更新、CI、Bot 内部ロジック (見え方が変わらないもの)
- 判別がつかないコミットは `git show <hash>` で本文と差分を確認

### Step 2: 1 行サマリへ書き換え

各コミットに対して **40〜60 字程度の日本語1文**に圧縮する。方針:

- バグ修正: 「<どんな症状> の不具合を修正」
- 機能・UI 変更: 「<対象> を <どう変えたか>」
- 内部用語(変数名・ファイル名・略語) は使わない。プレイヤー目線の言葉に置き換える
- 経緯・理由は省く。何が変わったかだけ
- 必要なら例を `（"Aフラ" → "Jフラ" など）` のように括弧で1つだけ添える。ただし、**コミットメッセージ内の具体値（特定のカード・特定の数値・特定のシナリオ）はその症状の一例にすぎないことが多い**ので、それを唯一の症状であるかのように書かない。一般化できるならまず一般化して書き、例示する場合は「など」「例えば」を明示する

### Step 3: ツイート文面を組み立てる

形式は固定:

```
【M/D アップデート ver.<commit-id>】
・<項目1>
・<項目2>
・<項目3>
```

ルール:

- タイトルは **`【M/D アップデート】`**。範囲指定で複数日にまたがる場合は `【M/D-M/D アップデート】`
- 箇条書きは **中黒 `・`** 始まり (ハイフン `-` は使わない)
- **絵文字なし、ハッシュタグなし、URL なし** (アップデート報告は素のテキストで出すのが既存スタイル)
- 1ツイート (140字) に収まらない場合は、ユーザー視点で重要度の低い項目をばっさり削る。「他N件」のような注釈は付けない
- 該当する変更が無ければ「対象期間に main 取り込みはありません」とだけ伝えて終わる

### Step 4: 出力

ツイート本文をコードブロック1つで提示する。直前に **対象範囲とコミット数** を1行、直後に **採用した各コミットのハッシュと元サブジェクト** を箇条書きで添える。例:

> 対象: 2026-05-07 の main 取り込み 3 コミット
>
> ```
> 【5/7 アップデート】
> ・…
> ・…
> ・…
> ```
>
> - `7aeedef` fix(ui): プリフォールド予約が…
> - `6da7bb4` fix(hilo): ショーダウン役名を…
> - `ac07fae` ui(plo): フラッシュ表示を…

ファイル保存・メモリ保存は **しない**(書き捨て)。

## 参考: 既知の良い文体

```
【5/7 アップデート】
・フォールド予約のチェックが残って次のターンで勝手にフォールドされる不具合を修正
・Hi-Lo ロー単独勝ち時の役名表示が崩れる不具合を修正
・PLO のフラッシュ表示をホール由来の最高ランクに変更（"Aフラ" → "Jフラ" など）
```
