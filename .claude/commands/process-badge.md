バッジ画像を加工するスキル。

## 入力
引数: `$ARGUMENTS`（元画像のパスと出力名をスペース区切り、または元画像のパスのみ）

例:
- `/process-badge public/images/badges/source.png penguin`
- `/process-badge public/images/badges/source.png` （出力名を聞く）

## 処理手順

1. 元画像を Read ツールで確認し、内容を把握する
2. 画像に含まれるバッジの数を判断する:
   - **1つ**: そのまま円形バッジとして切り出す
   - **複数（横並び）**: 画像を等分割し、それぞれ個別に切り出す
3. 出力名が引数にない場合、ユーザーに聞く（複数の場合は各バッジの名前を聞く）
4. Python (Pillow) で以下の加工を行う:
   - 画像を正方形にクロップ（高さベース、中央）
   - 明るさベースで円形バッジの境界を自動検出
   - 検出した円に合わせてタイトにクロップ
   - 円形の透過マスクを適用
   - 256x256 (`{name}.png`) と 512x512 (`{name}@2x.png`) で保存
5. 加工結果を Read ツールで表示してユーザーに確認
6. 出力先は元画像と同じディレクトリ

## 注意
- 元画像は加工後も残す（削除しない）
- 既存ファイルがある場合は上書きする
- 暗い背景に円形バッジがあるパターンを想定
