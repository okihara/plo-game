# プリフロップハンド評価（共有パッケージ）

フロント（例: `HandAnalysisOverlay`）とサーバー（Bot AI の入力）の双方で使う、プリフロップの「強さ」スコア。**数値・型の正は常にソース** `packages/shared/src/preflopEquity.ts`（および `preflopEquity.json`）に従うこと。本文の表は理解用であり、コードと食い違う場合はコードを優先する。

## スコア算出（概要）

1. **エクイティルックアップ**: 事前計算済みモンテカルロ結果（正規化ハンド × 反復）から 6 人テーブル想定のオールインエクイティを引き、0–1 に min-max 正規化する。
2. **プレイアビリティ補正**: エクイティ実現しやすさを、構造フラグで加減算する。

## プレイアビリティ補正（参考表）

| 補正項目 | 値 | 理由 |
|---------|------|------|
| ダブルスーテッド | +0.04 | フラッシュドロー2つでエクイティ実現◎ |
| シングルスーテッド | +0.02 | フラッシュドロー1つ |
| ランダウン（ダングラーなし） | +0.03 | ストレートドロー豊富 |
| ラップポテンシャル | +0.01 | ドロー力あり |
| Aスーテッド | +0.02 | ナッツフラッシュドロー保証 |
| ダングラー | -0.04 | 孤立カード、ポストフロップ不参加 |
| トリプルスーテッド | -0.03 | フラッシュアウツ減少 |
| レインボー | -0.02 | フラッシュドローなし |
| ペア+レインボー（ドローなし） | -0.03 | セット以外の発展性なし |
