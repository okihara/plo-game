# サイト流入動線の整理

## 6系統の流入経路

### 1. 能動的に探して来る
- 検索
- ブックマーク
- 直接URL入力
- 過去に見たことがあって再訪
- 「○○ 公式」「○○ 比較」みたいな指名/用途検索

### 2. SNS経由で流れてくる
- Xでフォローしてるアカウントのリツイート
- 有名人がすすめていた
- なんか話題になってる
- インフルエンサー投稿
- 友人のシェア
- Instagram、TikTok、YouTube、Threads、Facebook経由
- Reddit、はてブ、掲示板、コミュニティ投稿

### 3. 人から直接すすめられる
- リアル友人からのすすめ
- 仕事仲間からの共有
- 家族・知人からの紹介
- クチコミ
- オフラインで見聞きして後からアクセス

### 4. 外部メディアや他サイト経由
- 広告
- ニュース記事で紹介
- ブログ記事で紹介
- 比較サイト・まとめサイト
- レビューサイト
- 他サービス内のリンク
- メルマガ内リンク
- プレスリリース経由

### 5. プラットフォーム内レコメンド
- 「おすすめ記事」
- 関連リンク
- アプリストア・検索結果上位
- YouTube概要欄、プロフィールリンク
- サジェスト、ランキング、急上昇

### 6. すでに接点がある人の再訪
- ブックマークされてる
- 前に使ったことがある
- 会員登録済みでメールが届いた
- プッシュ通知
- LINE通知
- リマーケティング広告
- 定期巡回

---

## 施策向けの切り方

- 検索
- SNSで流れてくる
- 身近な人からの紹介
- 有名人・インフルエンサー紹介
- 話題化・バズ
- 広告
- 再訪導線（ブックマーク、通知、メルマガ）

---

## 新規流入 vs 再訪流入

### 新規流入
- 検索
- SNS
- 紹介
- バズ
- 広告
- 外部メディア

### 再訪流入
- ブックマーク
- メルマガ
- 通知
- 再指名検索
- リターゲティング広告
