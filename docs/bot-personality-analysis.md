# ボットパーソナリティ分析

## 概要

PLOボットは20種類のパーソナリティテンプレートを持ち、100体のボットに割り当てられる。
最初の20体は明示的に定義され、残り80体はボット名のハッシュ値 % 20 で決定的に割り当てられる。

## 好成績ボット分析

以下の4体が特にいい成績を上げている：

| ボット名 | 割り当てパーソナリティ | タイプ |
|---------|-------------------|------|
| **Sora_99** | YuHayashi | バランスTAG型 |
| **SoichiroT** | TatsuyaN | バランスやや攻撃的型 |
| **mayu_plo** | TatsuyaN | バランスやや攻撃的型 |
| **KeisukeS** | shun_pkr | LAG控えめブラフ型 |

### パラメータ比較

| パラメータ | Sora_99 | SoichiroT / mayu_plo | KeisukeS |
|-----------|---------|---------------------|----------|
| VPIP | 0.25 | 0.30 | 0.35 |
| PFR | 0.20 | 0.22 | 0.26 |
| PFR/VPIP | 0.80 | 0.73 | 0.74 |
| 3Bet | 0.09 | 0.09 | 0.10 |
| Cベット | 0.65 | 0.64 | 0.62 |
| アグレッション | 0.80 | 0.80 | 0.82 |
| ブラフ | 0.13 | 0.13 | 0.09 |
| スロープレイ | 0.12 | 0.09 | 0.10 |

### 成功要因

1. **PFR/VPIP比 0.73〜0.80** — 参加する時はほぼレイズ。リンプ少なくプリフロップで主導権
2. **適度なVPIP (0.25〜0.35)** — ハンド選別しつつタイトすぎない
3. **Cベット 0.62〜0.65** — プリフロップの主導権をポストフロップでも維持
4. **アグレッション 0.80〜0.82** — 高いがやり過ぎない絶妙なバランス
5. **控えめなブラフ (0.09〜0.13)** — 無駄なブラフでチップを失わない
6. **スロープレイ 0.09〜0.12** — 適度にトラップを混ぜる

## 設計指針（強化方針）

### 全体の目標レンジ
- VPIP: 0.20〜0.38（極端なニット/ルースを削減）
- PFR/VPIP: 0.60以上（リンプ抑制）
- Cベット: 0.50〜0.75
- アグレッション: 0.55〜1.00
- ブラフ: 0.06〜0.18（0.20超は抑制）

### 追加パラメータ（fold系）
- `foldTo3Bet`: 3ベットに対するフォールド頻度 (0.40〜0.70)
- `foldToCbet`: Cベットに対するフォールド頻度 (0.35〜0.60)
- `foldToRiverBet`: リバーベットに対するフォールド頻度 (0.40〜0.70)

PLOでは over-fold が大きなリークとなるため、これらを明示的にコントロールする。
