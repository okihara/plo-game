# モバイル UI レイアウト（9:16 縦画面）

メインゲーム画面の大まかなブロック構成。詳細は `src/pages/OnlineGame.tsx` や各コンポーネントを参照。

## ブロック構成（概念図）

```
┌──────────────────┐
│ ヘッダー (4%)     │  ← PLO | ブラインド | 設定
├──────────────────┤
│                  │
│   PokerTable     │  ← 楕円テーブル、6 人配置
│   (flex-1)       │     CommunityCards（中央）
│                  │     Player × 6（円周配置）
│                  │     Pot 表示（中央下）
├──────────────────┤
│ MyCards (24cqw)  │  ← 自分の 4 枚
├──────────────────┤
│ ActionPanel      │  ← プリセット + スライダー + 3 ボタン
└──────────────────┘
```

## プレイヤー配置（PokerTable）

- `humanIndex`（自分の席番号）を基準に 6 人を回転配置する。
- `positionIndex === 0` が画面下部（自分の位置）。
- `positionIndex !== 0` のプレイヤーのみ `Player` にカードを表示する（自分は `MyCards` で表示）。
